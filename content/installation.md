# Installation

## Prerequisites

- [Node.js](https://nodejs.org/) - at least v10.13 We recommend you have the latest LTS version installed.
- Install the latest version of [NuxtJs](https://nuxtjs.org) using npm.
  ```
  npm install -g nuxt
  ```
